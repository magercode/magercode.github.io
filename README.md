# The Mandiri Course

The Mandiri Course adalah situs dokumentasi dan arahan belajar mandiri. Situs ini **bukan** kelas atau platform dengan sesi pembelajaran.

Fokus utama:
- Memberikan peta arah belajar mandiri.
- Menyediakan contoh, ringkasan, dan referensi untuk belajar sendiri.
- Membantu mencari jalur belajar yang paling relevan berdasarkan kebutuhan.
